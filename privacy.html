Privacy Policy

Effective Date: February 25, 2026

Faaah! Twitter Like Sound ("the Extension") is a simple Chrome extension that plays a short "faaah" sound locally in your browser whenever someone likes one of your posts on X (formerly Twitter).

We take your privacy very seriously. This policy explains that the Extension does **not** collect, store, transmit, or share any user data.

1. Information We Do Not Collect

The Extension does **not** collect, store, log, transmit, or share any personal or sensitive information, including but not limited to:

  • Personally identifiable information (name, email, username, etc.)
  • Website content beyond transient observation needed to detect a like event
  • Web browsing history, user activity (clicks, scrolls, keystrokes), location, or any other behavioral data
  • Authentication information, financial data, health data, communications, or anything else

All functionality (sound playback) happens **entirely locally** in your browser. No data ever leaves your device or is sent to any server (including ours).

2. How the Extension Works (Limited Data Access)

To detect when someone likes your post, the Extension requires access to x.com and twitter.com pages (via host permissions in the manifest). It uses a content script to observe specific visible changes on your own profile, notifications, or tweet pages (e.g., like count increase or notification element).

This is strictly limited to:

  • Triggering a local audio file playback
  • No reading, storing, or using any tweet text, usernames, follower counts, or other content
  • No modification of page content
  • No logging or analytics
Limited Use Disclosure (Chrome Web Store Requirement)
Any transient access to website content adheres to Google's Limited Use requirements: the Extension only uses accessed data to provide the single user-facing feature (playing a sound on likes) as prominently described in the Chrome Web Store listing and UI. We do not:
  • Use or transfer data for advertising, re-targeting, or interest-based purposes
  • Allow human review of any data
  • Transfer data except as required by law or for security/abuse prevention
  • Use data for any purpose unrelated to the Extension's core function

3. Third Parties

The Extension does not use any third-party services, APIs, analytics tools, ads, or trackers. The sound file is bundled locally (or loaded from a static source with no tracking).

4. Security

Since no data is collected or transmitted, there are no external security risks related to user data from this Extension.

5. Changes to This Policy

If the Extension is ever updated to handle data differently, we will update this policy and note changes in the Chrome Web Store listing. Continued use after changes constitutes acceptance.
